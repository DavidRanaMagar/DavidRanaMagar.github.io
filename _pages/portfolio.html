User and Form management System Project

This is a web application developed using Django as the backend server, React for the frontend, and SQL as the database. The project was developed by a team of four. You can visit the repository to check out the code or run the project locally.

Features:

  • User login
  • Separate routes for admin and users
  • Data fetching and routes based on user type
